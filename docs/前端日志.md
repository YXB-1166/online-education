# 前端开发日志 — 在线教育辅助教学系统

## 技术栈
- Vue 3 + Vite 5 + Element Plus 2.5 + Pinia + Axios
- 项目位于 `frontend/` 目录，独立于后端开发
- 开发服务器端口 3000，热重载

## 项目初始化
- 创建 Vue 3 + Vite 5 项目
- 安装依赖：vue-router、pinia、element-plus、@element-plus/icons-vue、axios
- 配置 `vite.config.js`：
  - 设置 `server.host: '0.0.0.0'` 解决 Windows IPv4 浏览器无法连接 Vite 默认 IPv6 绑定的问题
  - 代理 `/api → http://127.0.0.1:8000`（指向 Gateway）
  - 端口 3000

## 路由与页面
- 登录页（Login.vue）
- 学生页面：我的课程、选课、论坛、提交作业
- 教师页面：我的课程、学生名单、选课审批、发布作业、批改作业、论坛管理
- 管理员页面：用户管理、课程管理
- 路由守卫：根据 role 字段（1=学生 2=教师 3=管理员）控制页面访问

## 核心功能
- **JWT 登录**：登录后在 Pinia store 保存 token + user 信息，Axios 拦截器自动添加 Authorization 头
- **选课审批流**：
  - 学生选课 → 状态 '0'（待审核）
  - 教师审批通过 → 状态 '1'（已选课）
  - 教师审批不通过 → 状态 '4'（审核不通过）
  - 学生退选 → 状态 '2'（已退选）
- **选课页面**：PendingApprovals.vue — 教师查看待审批选课列表，可批量通过/驳回
- **学生名单**：StudentList.vue — 教师查看某课程所有选课学生，状态标签用彩色 el-tag 显示（0-4）
- **分页**：el-pagination 组件配合后端 LIMIT/OFFSET 分页
- **论坛**：帖子列表 + 回复展示
